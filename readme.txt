Меню

[
	# | Помощь
	contact | Контакты
	page/support | Поддержка
]



Настройка галереи lightgallery
====================================================================================================

* Создайне новую страницу типа static
* Установите Обработчик HTML — Нет
* Выберите Профиль оформления — theme-lightgallery.css
* В тексте записи разместите html-каркас:

<div class="lightgallery">
<a href="адрес изображения"><img src="адрес миниатюры"></a>
<a href="адрес изображения"><img src="адрес миниатюры"></a>
<a href="адрес изображения"><img src="адрес миниатюры"></a>
</div>

* Если изображения много, то можно сгенерировать в Загрузках Код галереи, но использовать только коды [gal]...[/gal]. В этом случае блок миниатюр будет таким:

<div class="lightgallery">
[gal=адрес миниатюры]адрес изображения[/gal]
[gal=адрес миниатюры]адрес изображения[/gal]
[gal=адрес миниатюры]адрес изображения[/gal]
</div>
